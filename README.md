# 거상(모작)

### 프로젝트 개요

- 서비스 소개
	- 2D 게임 거상을 3D로 재구성한 프로젝트


- 개발 인원
	- 1명

- 개발 기간
	- 2020.05.01 ~ 2020.08.01 (3개월)


### 사용 기술
- 언어
	- JAVA, C#

- 프로토콜
	- HTTP, TCP/IP

- 데이터베이스
	- MariaDB

- 게임엔진
	- Unity

- 라이브러리
	- Netty, Flatbuffers, Jdbc, Gson
